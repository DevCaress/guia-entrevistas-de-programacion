## Ejemplo aplicado

Este ejemplo separa el calculo del reporte de su salida. En una entrevista, esa separacion ayuda a explicar una unica razon de cambio por clase.

```ts
type Order = {
  total: number;
  taxes: number;
};

class InvoiceCalculator {
  total(order: Order) {
    return order.total + order.taxes;
  }
}

class InvoicePrinter {
  print(amount: number) {
    return `Total: ${amount}`;
  }
}
```
